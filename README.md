## 구현 기능

> ### 메인 페이지

- 부트스트랩을 이용한 배너, 캐스팅, 프로필 캐러셀 구현
- 동영상 파일, 유튜브 영상 삽입
- 탑 스크롤 버튼 구현
  > ### 회원가입 페이지
- 회원가입 시 유효성 검사 구현
- 카카오 우편번호 API 사용
  > ### 캐스팅 페이지
- 카테고리 옵션 구현
- 북마크 구현
  > ### 오픈 프로필 페이지
- 프로필 모음 캐러셀 구현
- 북마크 구현
- 카테고리 옵션 구현
  > ### 프로필 상세보기 페이지
- > ### 마이 프로필 페이지
- 프로필 캐러셀 구현
- 프로필 편집 페이지 구현
  > ### 커뮤니티 페이지
- 고객 지원 페이지의 정보 & 팁 게시판 활용
  > ### 고객 지원 페이지
- 유저가 보고 싶은 것만 볼 수 있도록 FAQ를 아코디언으로 구현
- 이벤트를 한 눈에 확인하기 쉽도록 테이블 형태로 구현
- 문의하기를 누르면 작성 모달 창이 띄고, 제출 시 Sweet Alert 사용
  > ### 클라우드 서버 구축
- 네이버 클라우드 플랫폼을 이용해 웹 서버를 구축
